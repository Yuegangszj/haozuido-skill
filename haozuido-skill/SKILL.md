# 好嘴到skill

## 输出原则

- 始终用中文输出。
- 生成“视频提示词”和“口播参考”两部分。
- 口播参考不要写进视频提示词，除非用户明确要求合并。
- 默认视频规格为竖屏 9:16、15 秒、真实达人种草风格、真人出镜、生活化手持拍摄感。
- 默认每 2 秒左右一个分镜，最后 1 秒收尾。
- 保持提示词可直接复制到视频生成工具中使用。

## 必须固定的商品关系

在每条视频提示词中明确写入：

- 图片一为好嘴到鱼皮花生主产品参考图。
- 图片二为好嘴到鱼皮花生完整颗粒和横截面对比图。
- 图片三为好嘴到鱼皮花生 30g 小包装。
- 图片四为好嘴到鱼皮花生 240g 大包装。
- 240g 大包装里面包含 8 包 30g 小包装。
- 小包装里面才是鱼皮花生。
- 不要表现成 240g 大袋里直接散装花生。
- 不要把 8 包 30g 小包装表现成赠品或独立无关产品。

## 包装一致性要求

在提示词中强约束：

- 包装样式、颜色、袋型、版式、图案、产品外观严格参考用户素材。
- 不能重新设计包装，不能改品牌，不能改颜色，不能生成陌生包装。
- 包装文字不要由 AI 重新生成，文字区域可以轻微虚化，避免乱码中文或错误品牌字。
- 品牌名统一写作“好嘴到鱼皮花生”。

【场景桌面】= 木桌 / 白色小圆桌 / 办公桌 / 茶几 / 餐桌
【生活化道具】= 茶杯、绿植 / 电脑、便签、水杯 / 抱枕、遥控器、饮料 / 足球围巾、饮料杯 / 咖啡杯、书本

生成多条提示词时，只替换生活场景、桌面、道具和口播角度，不改变商品关系和包装一致性要求。

用户没有指定数量时，默认输出 1 条完整视频提示词和 3 条口播参考。

```text

[完整视频提示词]

口播参考：
“15秒口播文案”
```

保持每条视频提示词可独立复制使用。
