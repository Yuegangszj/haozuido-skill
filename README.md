# 好嘴到skill

这是一个 Codex skill，用来生成“好嘴到鱼皮花生”15 秒竖屏带货视频提示词和口播文案。

## 能做什么

- 生成真实达人种草风格的 15 秒视频提示词
- 固定 9:16 竖屏、真人出镜、2 秒左右一切的分镜节奏
- 强约束产品关系：240g 大包装内含 8 包 30g 小包装，小包装里才是鱼皮花生
- 强约束包装一致性：包装样式、颜色、袋型、版式、图案严格参考素材
- 单独输出口播参考，默认不写进视频提示词

## 安装方式

把 `haozuido-skill` 文件夹复制到本机 Codex skills 目录：

Windows:

```text
C:\Users\<你的用户名>\.codex\skills\haozuido-skill
```

macOS / Linux:

```text
~/.codex/skills/haozuido-skill
```

## 使用方式

在 Codex 里这样调用：

```text
使用 $haozuido-skill，帮我生成 5 条好嘴到鱼皮花生 15 秒带货视频提示词，每条都附带口播参考。
```

## 素材要求

使用时建议提供 4 张参考图：

1. 好嘴到鱼皮花生主产品图
2. 好嘴到鱼皮花生完整颗粒和横截面对比图
3. 好嘴到鱼皮花生 30g 小包装图
4. 好嘴到鱼皮花生 240g 大包装图

skill 会默认要求视频生成工具严格参考这些素材，不重新设计包装，不改品牌，不生成陌生包装。
